GIFRecorder



Description

Animated GIF image recorder. Recorded GIF will loop forever. This recorder is not supported on WebGL.



Public Functions

CommitFrame Commit a video frame for encoding.
CommitFrame Commit a video frame for encoding.
CommitSamples This recorder does not support committing audio sample buffers.
FinishWriting Finish writing and return the path to the recorded media file.


Constructors
